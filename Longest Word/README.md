# Objective
    Write a function that takes in a string and returns the largest word in the array.

# Example Returns
* yourfunction(Ready,steady,go!) will return **'steady'**.
* yourfunction('Ready[[[,steady,go!) will return **steady**.
* yourfunction('ABCd') will return **ABCd**.
